MoonReflect

【作成理由】
ダブルバッファリング、衝突判定の学習のため。

【開発期間】
１ヶ月

【コンセプト】
ダブルバッファリングを理解し、Canvasを使いこなす。
衝突判定を理解し、応用できるようにする。

【所感】
衝突判定がまだ甘い。変化する速度(移動距離)に対応することは割とできるようになったが、衝突判定自体がまだ慣れない。
ぶつかる向きの辺りがまだ理解できてない？壁との衝突はさっと実装できるが、中のブロックとの判定がいまいち...
今回もブロック自体は四角なのに判定は円で三平方使ってるし。だからたまに挙動がおかしい。

【これからの課題】
ダブルバッファリングの応用、ビット反転、マスクなどのテクニックを学びたい。
衝突判定はまだ基本的なところを頑張る。
